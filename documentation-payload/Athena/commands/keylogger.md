## Summary
Start or stop the keylogger

- Needs Admin: False
- Version: 1
- Author: @checkymander

### Arguments
#### action

- Description: The action to perform (start, stop)
- Required Value: True
- Default Value: None

## Usage

```
keylogger -action start
keylogger -action stop
```

## Detailed Summary
Starts or stops an in-memory keylogger that captures keystrokes on the target machine.
